## 设计指南

- **配色方案**: 主要使用蓝色(#336699)搭配灰色(#F0F0F0)背景，确保可读性
- **字体**: 系统默认字体，正文16px，标题20-24px
- **图标**: 使用统一的图标集，确保一致性
- **响应式**: 所有视图必须在桌面、平板和移动设备上正常工作
- **可访问性**: 遵循WCAG 2.1 AA标准，确保系统可被所有用户使用
